1.0 Introduction

 Back to Top

This generated project contains:

  • .java and .xml files that implement a Web service client
  • an Ant build file named build.xml which automates the deployment steps
  • a file named build.properties, which is used to parameterize the build.xml file

The component named LeanWSRunnerClient is implemented as a Web service client. The implementation of this Web service client is based on JAX-WS 2.1.1 RI. For the Web service to run the client, you must first correctly install it on WebSphere CE. To do so, follow the instructions in the readme.html file generated in the Web service project.

1.1 Prerequisites

You must have ILOG JRules 7.1 and WebSphere CE 4.2.3 installed.

You must first install the Web service and display the WSDL file in a browser before you can generate the .java file from the .wsdl file. These files are necessary for the client to compile.

Note:
The Web service is available on a default local WebSphere CE installation at http://localhost:8080/jaxws-RuleApp/leanWS?wsdl 

1.2 Key Points to Remember

  • Set your JAVA_HOME environment variable (JAVA_HOME is the folder where the Java(TM) 5 SDK is installed)
